🌍 Project Pitch: Nairobi Waste Tracker 💡

🌟 Introduction

Nairobi Waste Tracker is a smart solution designed to improve waste management in Nairobi by providing real-time tracking, reporting, and community involvement.

🌟 Why Nairobi Needs This

📌 The Problem: Nairobi struggles with inefficient waste management, leading to overflowing bins and littered streets.

✅ Introducing Nairobi Waste Tracker – A smart, real-time solution to track and improve garbage collection across the city.

🚛 The Problem We Face

  • ❌ Unreliable Garbage Collection – No timely updates on pickups.
  • 🗑️ Overflowing Bins & Litter – Poor management affects public health.
  • 🏡 Limited Community Involvement – No easy way to report waste issues.

💡 Solution

Nairobi Waste Tracker provides an interactive platform where residents can report waste hotspots, track bin status, and receive real-time updates on garbage collection schedules.

🔑 Key Features

  • ✅ Live Garbage Collection Schedules
  • ✅ Bin Status Updates
  • ✅ Report Waste Hotspots
  • ✅ Community Leaderboard
  • ✅ Dark Mode for User Experience

🛠 Technologies Used

  • 📂 Frontend: HTML, CSS, JavaScript
  • 📂 Data Storage: Local Storage for reports & leaderboard tracking

🚧 Challenges Faced

  • ❌ Dark Mode Toggle Issue – Initially, it didn’t apply styles properly.
  • ✅ Solution: Fixed JavaScript and CSS to ensure smooth transitions.
  • ❌ Live Wallpaper Crashes – Adding a video background caused performance issues.
  • ✅ Solution: Switched to optimized GIFs for better compatibility.
  • ❌ Spacing & Layout Issues – Elements were not well-aligned.
  • ✅ Solution: Improved CSS for better structure and responsiveness.
